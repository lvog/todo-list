# Todo List App

A simple Todo List web application that allows users to create, edit, delete, search, filter and reorder notes.  
The application stores all data in **localStorage**, so tasks remain saved after page reload.

## Features

- Add new notes
- Edit existing notes
- Delete notes
- Mark notes as completed
- Search notes
- Filter notes (All / Completed / Active)
- Drag and drop to reorder notes
- Popup form for adding/editing notes
- Empty state illustration when no notes exist

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

No frameworks or libraries were used.
